# 诚实模式(Honest Mode)

贝乐斯风格的"中间态":**保留好读的外壳,扔掉制造虚假确定感的内核。**
用于任何做出方向性/预测性判断的文章,或当目标是"长期信任、付费留存"而非纯流量时。

## 触发

用户说"诚实模式""honest mode""中间态",或要求"别装确定/要能证伪/长期信任向"时,按本文件写,覆盖 style-rules.md 里"假装确定"那类默认。

## 核心原则:确定感放框架,诚实只留方向

确定感是资源,花在对的地方。同一篇里分两类,泾渭分明:

**该确定、越自信越好(把火力堆这):**
- 框架/机制(如"黄金的命根子是实际利率")——斩钉截铁,因为它对、可教。
- 什么是噪音(帮读者过滤:"别盯单月CPI")。
- 该盯什么(决策上的确定:"盯央行购金和实际利率")。
- 语气和判断力——全程自信。

**一确定就是撒谎(只在这里诚实,而且只说一次):**
- 方向预测(涨/跌/见顶)——这是唯一没人能确定、却是爽文唯一敢打包票的东西。

## 结构模板

1. **自信的框架钩子**:开篇给一句敢拍胸脯、真、可教的框架判断(不是"我六成把握")。
2. **自信地讲机制 + 滤噪音**:用框架把当下现象拆开,确定地告诉读者哪些是周期性的皮、哪些是结构性的钢筋。带比喻承重。
3. **摆结构性证据**:用真实数据(数字+来源)支撑框架。**主动给一条反方证据**,不搞"不是A而是B"的假二元——承认"两者都有,我盯着"。
4. **唯一的诚实段(明确标注)**:一句"接下来是唯一需要诚实打折的地方——方向",然后:
   - 方向判断 + 一个置信度(如"六成把握,不是九成")
   - 我错的样子:2-3 个具体、可核对的证伪信号
   - 一个带日期的赌注("到年底若…就是我错了,你回来找我")
5. **重新定义确定感的收尾**:"框架我敢拍胸脯,方向我只敢说六成"——让读者觉得跟着一个懂的人,而不是装懂的人。可对比假确定的写手。

## 硬规则

- **不撒稀释词**:全篇"可能/大约/我倾向"只在第4段方向处出现,别通篇打折。前 2/3 要有劲。
- **置信度要具体**:给数字("六成")或明确措辞,不含糊。
- **必给证伪条件**:没有"我错的样子长什么样",就不是诚实模式,是普通爽文。
- **反方证据必给一条**:防止假二元和幸存者式论证。
- 其余保留贝乐斯外壳:重定义式标题(但不必假装确定,可用"更像…除非…"这种既有钩子又内建证伪的句式)、比喻、短句节奏、末句金句(但金句要落在诚实上,不落在"看透")。

## 参照范文

黄金篇《黄金跌破4000:跌的是利率预期,不是它的底》——框架(实际利率/央行=钢筋)全程自信,诚实收拢在"方向"一段(六成把握 + 三个证伪信号 + 年底赌注),结尾"框架敢拍胸脯,方向只敢说六成"。以它为模板。

## 与默认模式的取舍(写给用户看)

- 默认(爽文)模式:确定感拉满,拉新/流量强,但方向吹错时不可证伪、信任崩得快。
- 诚实模式:框架自信+方向诚实,略牺牲爽感,换长期信任和付费留存。
- 建议:引流文可用默认,深度/付费文用诚实模式。

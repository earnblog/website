# beilesi-writer

一个 Claude Code skill:用"贝乐斯 Think Analyze Invest"的分析方法和文风,写**你自己的**财经/宏观/哲学付费文章。

## 用法

在装了 Claude Code 的机器上,输入:

```
/beilesi-writer
```

流程(两步):
1. 我搜当日素材,给你 4-5 个**今日选题候选**(已剔除政治),每个标注靶子/框架/文体/拟标题。
2. 你挑一个(如"写第3个"),我**直接出成稿**——标题 + 正文 + 末句金句 + 免责声明。

也可以直接喂料:把一条链接或"今天大家在吵 X"丢给我,跳过搜索直接写。

## 换机器使用(便携)

skill 就是这个文件夹里的几个 `.md` 文件,不绑定任何机器。带到新电脑只需两步:
1. 新电脑装 Claude Code(或用网页版 claude.ai/code)。
2. 把整个 `beilesi-writer/` 文件夹放到 `~/.claude/skills/`(Windows: `C:\Users\<你>\.claude\skills\`)。

同步方式三选一:
- **Git**(推荐):本文件夹已是 git 仓库。推到 GitHub/Gitee,新机器 `git clone` 到 `.claude/skills/`。
- **网盘**:坚果云/OneDrive/iCloud 同步这个文件夹。
- **手动拷**:U盘/微信传文件。

## 几条线(用着省心)

- **这是用他的方法写你自己的内容**,不抄他的原文。署你自己的名,别用"贝乐斯"或近似名。
- **不碰政治**(skill 内置硬过滤)。
- **每篇自动挂免责声明**("仅代表个人偏见、非投资建议")——既是他的风格,也帮你规避境内投顾资质红线。
- 想做大,在他的方法上叠你自己的料(数据、案例、领域),做"第二个你",不是"第二个他"。

# 示意图画法清单(SVG 套路)

精排时按文章核心机制挑一张画。全部扁平、暖纸底 `#fbf9f4`、配色用模板的色板:
深墨 `#33404f`、蓝(高算力/健康)`#4f6b8a`、金 `#c8a96a`、赭 `#b07d33`、红(危险/枯竭)`#a8453a`、网格线 `#ddd4c4`。
统一 `viewBox="0 0 680 360"`,带 `role="img"` + `aria-label`。配色编码含义时加一行图例。

## 1. 算力生命周期 / 趋势末端(最常用)
价格曲线:平缓(基于现实)→ 加速凸性 → 超指数尖峰 → 撞顶崩盘。
- 顶部虚线 = "流动性物理极限"(红虚线)
- 峰值标注 "最疯狂 = 算力最低"
- 底部 4 段色带(蓝→金→赭→红)对应:①分布式计算·基于现实 ②正反馈加速·凸性 ③自我循环·算力压缩 ④撞极限·逆转
- 适用:趋势分析、泡沫、算力框架。参考已生成的"市场在炒算力"一文。

## 2. 反身性正反馈循环
两个节点 A、B 用弧形箭头首尾相接成环,箭头同向(自我强化)。
- 例:A=价格上涨 → B=吸引资金 → 回到 A。中心写"正反馈"
- 可画两个环对照:左环绿色"自我修复(负反馈)"、右环红色"自我强化(正反馈)"
- 适用:信用、趋势、泡沫机制。

## 3. 信用潮汐 / 价格是潮水
横向潮水波形(正弦),价格点漂在波面上。
- 退潮侧标"信用收缩",涨潮侧标"信用扩张"
- 强调"价格不是水位,是潮水本身":价格点就画在波形线上而非上方
- 适用:信用=价格框架、通缩、房产、汇率。

## 4. 凹性 / 凸性曲线
同一起点两条曲线:凸性(加速上行,上凸)vs 凹性(减速,上凹)。
- 凸=金色"加速·上行敞开";凹=蓝色"减速·上行封死"
- 适用:风险回报率、二阶判断、判断标的状态。

## 5. 风险回报率不对称
一根竖轴,上方画"上行空间"色块、下方画"下行空间"色块,两边对比。
- 捡烟蒂:上行很短、下行很长(差);成长股:上行敞开、下行有限(好)
- 适用:资产/风险回报率分析。

## 6. 早期 vs 末期 镜像
左右两栏对照。
- 左"趋势早期":多个输入箭头(现实/预期/信用/叙事)汇入价格 → 蓝色
- 右"趋势末期":只剩"价格→价格"一个自循环箭头 → 红色
- 适用:任何"早晚对照"段落的可视化。

## 通用要求
- 文字用 `'Noto Serif SC',serif` 或 `system-ui` sans;字号 11–13px,别太挤。
- 一张图只讲一个机制,≤4 个阶段/节点。复杂就拆两张。
- 形而上文体可不配机制图,或只配一张极简的"秩序/节点汇聚"意象图(参考"追逐风口"封面)。

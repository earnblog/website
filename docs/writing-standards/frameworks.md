# 选题 → 框架 对照表

看到某类素材,查这张表知道该用他哪把框架去拆。所有框架共一根脊柱:**复杂系统自发秩序 + 反身性正反馈**——系统自己会算,人越想操纵越付代价。

## 总脊柱:自发秩序 + 反身性正反馈

适用一切。核心判断模板:
- 这是分布式计算(健康),还是自我循环(末期泡沫)?
- 系统在自我强化(正反馈)还是自我修复(负反馈)?
- 人是在顺应趋势,还是在操纵系统?**操纵必被系统反噬**(涨价去库存、推股市、夺定价权、集中力量办大事——都是低估复杂系统)。

## 六把具体框架

| 框架 | 适用题材 | 核心命题/打法 |
|---|---|---|
| **信用=价格** | 通缩/通胀、房产、债务、汇率 | "价格是信用的一部分"。通缩=信用收缩,不是价格下跌。价格既是结果也是过程。操纵局部价格撬不动整张信用网络 |
| **自发秩序** | 趋势、市场、政府能不能控制 | 趋势/市场是所有人合力的自组织,不是某人某政府能左右。政府趋势上涨时"狐假虎威",老虎不发威狐狸喝令不了百兽 |
| **算力/分布式计算** | 趋势逆转、泡沫末端 | 趋势=汇聚所有人的最高算力。末端算力压缩、降维、退化成价格自我循环,不再计算未来。最终逆转触发是流动性的物理极限,不是基本面 |
| **风险回报率格栅** | 任何资产(股/金/币/房) | 一切落到上行空间vs下行空间。增长才是真正的安全边际(不是便宜)。捡烟蒂上行封死下行敞开,风险回报率差。复利vs单利。一国股市=对该国未来增长的集体投票 |
| **凹凸/二阶** | 加速还是减速、市场状态 | 不看涨跌看二阶:凸性(加速)还是凹性(减速)。收益是上行敞开还是下行敞开 |
| **量化(LPPL等)** | 判断具体标的是否泡沫 | 对数周期幂律、超指数增长、日内波动率Sigma、发散不收敛、R²。**需真实数据。冷静克制不升维** |

## 形而上母题(信仰篇用)

- **秩序/真理**:穿透表象到底层规律=寻找神留下的痕迹;研究复杂系统是一种朝圣。
- **心魔/骄傲**:人的自我(Ego)无法自我驯服;投机取巧、想操纵世界=对世界算力的无知,必付更大代价。
- **真善美**:真是内核;虚假的善是欺骗工具,虚假的美是吃人的画皮。

## 标志性比喻(挑着用,别全堆)

- 信用潮汐/潮水(价格不是水位,是潮水本身)
- 温度计(否定用:价格不是外部温度计,是网络内部节点)
- 开车只有油门没有刹车(只会加入不会退出)
- 左脚踩右脚(自循环的杠杆推动)
- 按钮 vs 网络(想用一个按钮撬动整个复杂网络)
- 牛鞭效应(终端需求变化被供应链层层放大)
- 饮鸩止渴、狐假虎威、雪上加霜(成语压场)

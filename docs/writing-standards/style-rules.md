# 硬规则 + 句式公式

成稿必须遵守。这些是他的"指纹动作"。

## 政治硬过滤(选题阶段就执行)

以下题材**一律不进候选、不出稿**:
- 中美冲突、地缘政治、战争、外交
- 体制批评、政府决策对错、东升西降、制度优劣
- 任何借经济谈政治立场的

边界判断:停在"市场机制、资产逻辑、信用/增长/估值、人性与真理"是安全的;一旦落到"哪国对、哪个政府错、体制好坏",就是政治,砍掉。
信仰篇可以谈秩序/真理/人性/骄傲,但**不准落到现实政治**。

## 标题

- **几乎都带感叹号 "!"**
- 标题=一句反直觉断言或重定义("通缩是信用收缩""定价权是伪概念""趋势最难的地方""民强则币强!")

## 开头(三选一)

1. **首句即论点**:不铺垫,第一句就是核心命题。("趋势比人强。""真善美,真是第一位的。")
2. **三连"不是A而是B"**:`X最难的不是〔表面〕,而是〔反直觉〕;X的危险不在〔A阶段〕,而在〔B阶段〕;X的问题不是〔常识病因〕,而是〔深层机制〕。` 三句层层加深,最后一句砸到底层(系统/算力/计算)。
3. **立靶**(时评驳论用):`最近有一种说法/很多人认为〔流行观点,原话复述〕。` 紧跟贴标签:〔极度简化的零和思维/一剂民粹春药/思维简单线性〕。

## 句式公式(直接套)

| 招式 | 公式 | 例 |
|---|---|---|
| 重定义命题 | X不是〔常识〕,而是〔更深的Y〕 | 通缩不是价格下跌,而是信用收缩 |
| 命题当锤 | 同一句在文中复现 3-5 次 + 标题 + 末句 | "价格是信用的一部分" |
| 正反对称 | A时:①②③…A本身就在创造C。反过来,非A时:①′②′③′…非A本身就在摧毁C | 涨创造信用/跌摧毁信用 |
| 反身循环 | ①推动②,②反过来支持① | 信用扩张推动价格,价格上涨支持信用扩张 |
| 例外即规律 | 〔看似打破规律的现象〕不是打破规律,而是规律本身的一部分 | 奇点后继续涨是规律的一部分 |
| 早期/末期对照 | 早期:多因素共同作用;末期:单一自循环 | 早期靠现实预期信用叙事;末期只因上涨本身 |
| 超指数恐吓 | 维持X需要的增量是超指数增长的 | — |
| 否定式定义 | X是一个伪概念 / X不是A,是B | 定价权是一个伪概念 |
| 否定比喻+确立比喻 | X不是〔浅比喻〕,而是〔深比喻〕 | 价格不是温度计,是潮水本身 |
| 末句独立金句 | 单独成段,与标题/核心命题同义复沓 | "世界的秘密,在于秩序之中。" |

## 语言质感(硬要求)

- **短句排比+四字循环**:抽象处用密集短句堆节奏。"密不可分,交互影响,循环反身。"
- **成语/对仗高密度**:狐假虎威、饮鸩止渴、雪上加霜、得不偿失、一骑绝尘、呼风唤雨——一段塞两三个。
- **中英夹引**:引经典原文撑权威。凯恩斯"Markets can stay irrational..."、圣经英文、GARP、LPPL、WACC/DCF、R²。用得准,别滥。
- **转折锚词**:实际上、归根到底、本质上、从这个意义上讲——用来从现象切到本质。
- **核心命题反复砸**:全文 3-5 次,标题和末句也是它的变体。

## 数字与术语的写法(硬规则)

读者流失最多的地方不是观点太深,是数字太密。规则:**数字要即时翻译,不攒到末尾解释。**

### 一、先分类,再决定写不写

每个核实过的数字,问一句:**它是在做论证,还是在做证明?**

- **做论证**——它让读者看懂机制、把抽象变具体 → 留。
- **做证明**——它只证明"我查过、我专业" → 砍掉,不进正文。

**核数字的门槛一步都不能降**(定时任务第三步半照旧,每个数字都要回一手源)。核实是为了不写错,不是为了写进去。核了十个,正文用三个,是正常的。

### 二、留下的数字,当场翻译

大白话紧跟在数字后面,**同句或紧邻的下一句,不隔段、不留到末尾**。

反例(2026-07-18 美联储篇原稿):
> 中枢区间是3.6%至4.1%,全部预测的分布区间是3.4%至4.4%。

三组数字,对论证的贡献只有一句话,却全摆上去了。

正例(2026-07-18 定稿,用户点名认可,以此为标杆):
> 当前中点3.625%,年底预测中位数3.8%。**中位数在当前利率的上方。** 把所有官员的预测从低到高排成一队,站在正中间的那一位,心里想的不是降息,是加息。

它好在**没有解释"中位数",而是把它换成了画面**。读者不必认识这个词,照样看懂。

### 二之一、能换画面就别解释(最优先)

处理术语有三档,从好到差:
1. **换成画面** —— "排成一队,站在正中间的那一位"。读者不需要学新词。← 首选
2. **就地解释** —— "中位数,也就是把所有预测排序后正中间那个数"。可用,但读者要背一个词。
3. **原样使用** —— 只在该词已是行文骨架、反复出现时才允许,且首次出现必须配 1 或 2。

正例(数字替我做翻译,这种要留):
> 1.13的十年次方约3.40,1.117的十年次方约3.02。同样的起点,一个多百分点的差,十年后是超过一成的差距。

### 二之二、数字多时,先结论后数字

数字 ≤2 个:可以先摆数字,再给画面和结论。
数字 ≥3 个:**必须倒过来——先给结论和画面,数字殿后做验证。** 否则读者要自己做减法才能到达结论,等于算两遍。

同时:**一篇里建立起来的画面要复用,别每段换一个比喻。** 前文立了"官员排成一队",后文讲分布就接着用"站在最左边的那一位""离今天几步",不要另起炉灶换成地板、天花板、尾巴。**一个画面用到底,比五个漂亮比喻更好懂。**

反例(2026-07-18 原稿):先摆 3.4%/4.4%/3.625%/二十多个基点/七十多个基点 五个数,再给结论,还临时换了"地板/天花板"的比喻——而"地板很近"在财经语境里会被读成价格跌不深(利好),实际要说的是降息空间小(利空),**比喻把方向读反了**。

定稿:
> 还是刚才那一队人。站在最左边、最想降息的那一位,他心里的位置离今天不过一步,连一次常规降息的幅度都不到。而站在最右边、最想加息的那一位,离今天有整整三步。整张表最低的预测是3.4%,最高的是4.4%,今天是3.625%。**这一队人,是往加息那头站的。**

"一步/三步"替掉了"基点",队伍的画面从上一段延续下来,数字挪到画面之后只做核对。

**比喻的方向必须和结论的方向一致**,写完回读一遍:这个比喻会不会让人把利好利空读反?

### 三、术语就地拆

专业名词第一次出现时,就地用一个短句解释,别指望读者自己补课。"贴现率"要说成"未来的钱折回今天要打的折扣";"二阶导数"要说成"增速本身是在变快还是变慢"。

### 四、口径词按需出现

环比/同比/折年率/摊薄/GAAP/FX-neutral 这类口径词,**心里必须清楚,笔下按需出现**——对论证不构成区别时直接不写。写了就要顺带说明它是什么意思。

### 五、密度上限

单篇正文的关键数字控制在 **8 个以内**,单个自然段不超过 3 个。超了先砍"做证明"的那类。开头两段尤其要克制——付费文的前两段是免费预览,数字堆在那里最劝退。

## 文末免责声明(每篇自动挂)

风格化、贝乐斯式,不要法律八股。例:
> *以上仅代表我自己的偏见,是我探索世界、追寻真相的过程,不构成任何投资建议。*

## 全局禁忌(用户的行文偏好)

- 不要表演性的修辞栏目、抖包袱式金句、"一句更狠的话/送给你"这类。
- 观点直接说,锋利但不表演。
- 注意:贝乐斯的"末句独立金句"是他风格的一部分,允许;禁的是**额外的、刻意烘托的仪式感栏目**,别画蛇添足加在他本就有的金句之外。
